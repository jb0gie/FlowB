# FlowB 🤖

> Your companion for discovering events — now in 3D

**[Live Site](https://jb0gie.github.io/FlowB/)**

## What is this?

This is FlowB — a small robot with a monitor for a head, here to help you discover events, form crews with friends, and coordinate your social calendar.

FlowB aggregates side events, hackathons, and meetups into one platform. Check in together, earn crew points, and never miss out on the action.

This 3D showcase brings FlowB to life with an interactive avatar you can engage with directly.

## Features

- 🤖 Interactive 3D robot avatar (FlowB with his signature screen-face)
- 📺 Toggle screen mode — watch his eyes move aside to reveal content
- ✨ CSS3D screen display for dynamic HTML content
- 🔔 Animated notification ring with subtle UV pulse
- 🎨 Dark theme with cyan accent glows
- 🎭 Smooth morph target animations (Idle loop, blinking, screen transitions)
- 🚀 Built with Three.js, TypeScript & web components

## Tech Stack

- **Framework:** Vite + TypeScript
- **3D Rendering:** Three.js + CSS3DRenderer
- **Avatar:** Custom GLB model with morph targets
- **Styling:** Shadow DOM + CSS custom elements

## Running Locally

### pnpm (recommended)
```bash
pnpm install
pnpm dev
```

### npm
```bash
npm install
npm run dev
```

Then open http://localhost:3000
